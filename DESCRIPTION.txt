4) EXTENDED DESCRIPTION (GitHub About / website meta / project submission)
--------------------------------------------------------------------------------
AgroScoreAI is a wheat-focused agri-fintech platform designed for Pakistan's smallholder farmers. The platform converts real farm activity — crop cycles, irrigation patterns, climate exposure, and transaction behavior — into an explainable AI credit score.

Unlike traditional lending systems that require formal documents, AgroScoreAI builds a digital financial identity from agricultural data. Farmers receive localized advisory in four languages (English, Urdu, Pashto, Balochi), while banks and microfinance institutions get a decision-ready risk view through an institutional dashboard.

This repository contains a fully interactive mobile-first frontend prototype with live AI scoring simulation, multi-language support, role-based workflows (Farmer, Bank/MFI, Field Officer), and an investor demo mode — no backend required.

Built as the Financing New Ventures (FNV) final project at FAST School of Management, Spring 2025.
